# Jarvis Trader - Expo Mobile App

This is a minimal Expo React Native starter for Jarvis Trader.

Quick start:

1. Install Expo CLI: `npm install -g expo-cli` or use `npx expo`.
2. From this folder run: `npm install` then `npm start`.
3. Use Expo Go or emulator to open the app.

Notes:
- This app demonstrates a basic UI and will later integrate with the Jarvis API endpoints.
